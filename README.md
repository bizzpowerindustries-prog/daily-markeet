# DailyMart.pk - Complete E-commerce Marketplace

## Overview
DailyMart.pk is a complete e-commerce marketplace platform like Daraz.pk, built with Laravel 9, Kotlin (Android), and Firebase.

## Features
- Multi-vendor marketplace
- Firebase Authentication
- Multiple payment gateways (Payfast, RapidGateway, Simpaisa)
- Wallet system for customers and sellers
- Flash sales and coupons
- Product management
- Order tracking with Pakistan Post integration
- Admin panel with full control
- Android apps for customers and sellers
- Real-time notifications
- SEO optimized

## System Requirements
- PHP 8.1 or higher
- MySQL 8.0 or higher
- Apache 2.4 or higher
- Redis (for caching and queue)
- Beanstalkd (for queue)
- Node.js 16+ (for frontend assets)
- Composer 2.0+
